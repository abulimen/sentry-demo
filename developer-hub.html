Testing Environment (Sandbox)

We provide a dedicated sandbox environment that mirrors our live production environment. This allows you to develop and test your integrations thoroughly without affecting live data or making real Pi transactions.

Features of the Sandbox:

  • Full API functionality.
  • Test Pi accounts with faucet capabilities (to simulate transactions).
  • Separate API keys for sandbox and production.
