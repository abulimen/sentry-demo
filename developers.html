Example: Create a Pi Payment Link

Here's a quick glimpse of how easy it is to use our Sentry API. This example uses our Python SDK.

# Python Example: Create a Sentry Payment Link for Pi
import sentry_pi_sdk # Hypothetical SDK name

# Initialize the SDK with your API Key
sentry_pi_sdk.api_key = "YOUR_SENTRY_API_KEY" 

try:
    payment_details = {
        "amount_pi": 10.5, # Amount in Pi
        "description": "Payment for Order #S12345",
        "merchant_order_id": "ORDER-S12345",
        "success_url": "https://yourstore.com/payment_success",
        "cancel_url": "https://yourstore.com/payment_cancelled"
    }
    payment_link_response = sentry_pi_sdk.PaymentLink.create(payment_details)
    print(f"Payment Link URL: {payment_link_response.url}")
    print(f"Expires At: {payment_link_response.expires_at}")
except sentry_pi_sdk.errors.APIError as e:
    print(f"An API error occurred: {e}")
