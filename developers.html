Example: Create a Payment Link

Here's a quick peek at how easy it is to use our API. (Illustrative example)

# Python Example
import sentry_sdk

sentry_sdk.init(api_key="YOUR_API_KEY")

payment_link = sentry_sdk.create_payment_link(
    amount=10.0,
    currency="PI",
    description="Payment for Order #12345",
    redirect_url="https://yourstore.com/payment_success"
)

print(payment_link.url)
